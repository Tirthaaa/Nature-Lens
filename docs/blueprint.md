# **App Name**: Nature Lens

## Core Features:

- Camera Integration: Real-time camera view for capturing plant images.
- Plant Identification: Analyze plant images to identify species and provide detailed descriptions using a generative AI tool.
- Detailed Plant Information: Display comprehensive information about the identified plant, including scientific name, habitat, species, and lifespan.
- Image Upload: Image upload feature to allow users to identify plants from existing photos.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke nature and growth.
- Background color: Light beige (#F5F5DC) for a natural, earthy feel.
- Accent color: Earthy orange (#D38A2E) to highlight important elements and calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet accessible reading experience.
- Use nature-themed icons to represent different plant categories and features.
- Clean and intuitive layout with clear sections for camera view, plant information, and image uploads.
- Subtle transitions and animations for a smooth and engaging user experience.
